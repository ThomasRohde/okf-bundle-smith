# Concepts

## Concepts

* [Open Knowledge Format](open-knowledge-format.md) - Concept graph format that represents durable knowledge as Markdown files with YAML frontmatter and links.
* [Concept File](concept-file.md) - Atomic Markdown document that represents one durable idea, asset, process, decision, source, or system in an OKF bundle.
* [Frontmatter Schema](frontmatter-schema.md) - Minimal metadata contract for OKF concept files and the recommended fields used by OKF Bundle Smith.
* [Reserved Files](reserved-files.md) - Index and log files that support navigation and change history without becoming concepts themselves.
* [Concept Links](concept-links.md) - Markdown links that connect OKF concept files into a traversable knowledge graph.
